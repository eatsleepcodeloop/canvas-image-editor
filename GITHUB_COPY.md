## About This Project

```text
This project is a standalone browser-based canvas image editor rebuilt from a production feature in a larger web application. I extracted the image-editing workflow, removed application-specific dependencies, and refactored it into a smaller portfolio-ready project focused on frontend interaction, canvas tooling, and clean presentation.

It demonstrates practical feature extraction, frontend refactoring, Fabric.js canvas work, UI state management for multiple editing tools, and converting a real product feature into a clear public portfolio project.
```
